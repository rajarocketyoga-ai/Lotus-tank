# LotusTank Manufacturing & Sourcing Research

## 1. Overview
This document compares two primary paths for bringing LotusTank's premium yoga tops to market: Local Print-on-Demand (POD) for rapid prototyping and design validation, versus Ethical Overseas Manufacturing for sustainable growth and healthy margins.

---

## 2. Path A: Print-on-Demand (POD)
**Best for: Initial design validation and "limited edition" testing.**

| Feature | Details |
| :--- | :--- |
| **Primary Candidate** | **Printful** (All-Over Print Yoga Tank Top) |
| **Fabric** | 82% polyester, 18% spandex. 6.78 oz/yd² (230 g/m²). |
| **Performance** | Four-way stretch, moisture-wicking, smooth/comfortable microfiber yarn. |
| **Cost per Unit** | ~$22.00 - $25.00 |
| **MOQ** | 0 (Zero inventory risk) |
| **Lead Time** | 2-5 days for fulfillment |
| **Pros** | No upfront cost; instant shipping; easy to test graphics. |
| **Cons** | Low margins (~50% vs 75%+ goal); limited fit customization; fabric may feel "standard" vs "premium". |

---

## 3. Path B: Ethical Small-Batch Manufacturing (Bali/Vietnam)
**Best for: Official Launch and Scaling.**

| Feature | Details |
| :--- | :--- |
| **Primary Candidate** | **Bali Swim** (Bali, Indonesia) or **Social Fabric** (Vietnam) |
| **Fabric** | Custom sourced technical fabrics (e.g., Recycled Nylon/Spandex, Italian Carvico). |
| **Performance** | Custom-engineered for hot yoga: ultra-wicking, anti-odor, compression-fit. |
| **Cost per Unit** | ~$10.00 - $14.00 (Target: **$12.00**) |
| **MOQ** | 50-100 units per style |
| **Lead Time** | 4-8 weeks (Sampling + Production) |
| **Pros** | High gross margins (~75%); custom fit/branding; sustainable/recycled options; premium "hand-feel". |
| **Cons** | Upfront capital required; higher logistics complexity; longer lead times. |

---

## 4. Cost & Margin Analysis (Target Retail: $48.00)

| Metric | Path A (POD) | Path B (Overseas) |
| :--- | :--- | :--- |
| **Manufacturing Cost** | $23.00 | $12.00 |
| **Shipping/Duty** | $5.00 | $4.00 (split over bulk) |
| **Total COGS** | $28.00 | $16.00 |
| **Gross Profit** | **$20.00 (41%)** | **$32.00 (66%)** |

---

## 5. Recommended Strategy: The "Hybrid Launch"
1. **Phase 1 (Month 1):** Launch 3 designs via **Print-on-Demand** to "Early Access" waitlist. Collect feedback on graphics and general interest.
2. **Phase 2 (Month 2-3):** Based on top-performing POD designs, place an order for 200 units (50 per style/color) with **Bali Swim**.
3. **Phase 3 (Ongoing):** Transition POD to "Experimental Drops" and Bali manufacturing to "Core Essentials".
